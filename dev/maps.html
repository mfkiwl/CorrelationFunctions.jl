Correlation Maps

CorrelationFunctions.Map.l2 — Function
l2(image; periodic = false)

Calculate L₂ (lineal path) correlation function map for the 1,2,3-dimensional image and return a CFMap object.

The image contains the probability of the voxel being in the correct phase.

For L2, only probabilities 0 and 1 are currently implemented.

Examples

julia> l2([1 0; 0 1]; periodic=true).result
3×2 Matrix{Float64}:
 0.0  0.5
 0.5  0.0
 0.0  0.5
